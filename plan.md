# 프로젝트 계획서: YouTube Info Viewer

본 프로젝트는 **PyQt5**와 **yt-dlp**를 사용하여 유튜브 영상의 URL을 입력받아 해당 영상의 기본 정보를 조회하는 데스크톱 애플리케이션을 구축하는 것을 목표로 합니다.

## 1. 개요
- **목적**: 유튜브 영상의 메타데이터(제목, 조회수, 썸네일)를 빠르게 확인하는 도구 제작
- **개발 언어**: Python 3.x
- **주요 라이브러리**: 
    - `PyQt5`: GUI 구성
    - `yt-dlp`: 유튜브 데이터 추출
    - `requests`: 썸네일 이미지 다운로드 및 로드

## 2. 주요 기능 (v1.0)
- **URL 입력**: 유튜브 링크를 입력할 수 있는 `QLineEdit` 위젯 제공.
- **정보 조회**: '조회' 버튼 클릭 시 `yt-dlp`를 사용하여 데이터 파싱.
- **데이터 시각화**:
    - **제목**: 영상의 제목 표시.
    - **조회수**: 천 단위 구분 쉼표가 포함된 조회수 표시.
    - **썸네일**: 영상의 썸네일 이미지를 `QLabel`에 표시.
- **로딩 피드백**: 조회 중 로딩 상태 메시지 표시로 사용자 경험 향상.

## 3. UI 레이아웃 설계
- **상단 (입력부)**: 
    - URL 입력창 (텍스트 박스)
    - 조회 실행 버튼
- **중앙 (이미지부)**: 
    - 썸네일 이미지 출력창 (고정 비율 유지)
- **하단 (정보부)**:
    - 영상 제목 레이블 (강조 텍스트)
    - 영상 조회수 레이블
    - 상태/로딩 메시지 레이블

## 4. 단계별 구현 계획

### Phase 1: 개발 환경 준비
- [ ] Python 가상환경 구성
- [ ] 필요 패키지 설치: `pip install PyQt5==5.15.10 yt-dlp requests==2.31.0`
- [ ] `requirements.txt` 작성 (버전 고정 포함)

### Phase 2: 기본 UI 구성
- [ ] `QMainWindow`를 상속받은 메인 클래스 작성
- [ ] 레이아웃 매니저(`QVBoxLayout`, `QHBoxLayout`)를 이용한 위젯 배치
- [ ] 기본 스타일(폰트 크기, 여백) 설정
- [ ] 조회 중 로딩 상태 메시지 레이블 추가 (예: "조회 중...", "완료", "오류")

### Phase 3: 데이터 추출 로직 (yt-dlp 연동)
- [ ] `yt-dlp`의 `YoutubeDL` 클래스를 활용한 정보 추출 함수 작성
- [ ] `extract_info(url, download=False)` 옵션을 사용하여 메타데이터만 수집
- [ ] 제목, 조회수, 썸네일 URL을 딕셔너리 형태로 반환
- [ ] **`QThread`를 활용한 비동기 처리**: 네트워크 요청을 별도 스레드에서 실행하여 UI 블로킹 방지

### Phase 4: 이미지 로딩 및 데이터 바인딩
- [ ] `requests`를 통해 썸네일 URL로부터 이미지 데이터(Bytes) 수신
- [ ] `QPixmap`과 `loadFromData`를 사용하여 이미지 표시
- [ ] 추출된 텍스트 데이터를 UI 레이블에 매핑
- [ ] 조회 완료 후 상태 메시지 업데이트

### Phase 5: 예외 처리 및 마무리
- [ ] 유효하지 않은 URL 입력 시 경고창(`QMessageBox`) 띄우기
- [ ] 네트워크 연결 오류 또는 정보 추출 실패 시 에러 메시지 표시
- [ ] 스레드 종료 처리 (앱 종료 시 안전하게 스레드 중단)

## 5. 프로젝트 구조
```text

├── main.py              # 프로그램 실행 메인 코드

```

## 6. 주요 기술 결정 사항
| 항목 | 결정 | 이유 |
|------|------|------|
| 비동기 처리 | `QThread` | UI 블로킹 없이 네트워크 요청 처리 |
| 이미지 로딩 | `requests` + `QPixmap` | 별도 파일 저장 없이 메모리에서 직접 표시 |
| 버전 고정 | `requirements.txt`에 명시 | 환경 재현성 보장 |
